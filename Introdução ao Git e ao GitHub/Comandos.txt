git init (Inicia um repositório)
git add (Mover arquivos)
git commit (Cria o primeiro commit)
git clone + URL (Clona repositório desejado)
git remote add origin + URL do repositório
git push origin master (Empurra arquivos alterados)
git add <file>...(Salva arquivos alterados) ou
git add . (Salva todos arquivos alterados)
git commit -m "Mudança feita"(Mensagem/Boas práticas)
git push origin main (Empurra arquivos alterados)
